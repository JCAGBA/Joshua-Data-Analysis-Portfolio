Back to Portfolio

Project Overview

This project studied smartphone addiction patterns, examining the impact of excessive phone usage on productivity and well-being. The aim was to provide actionable recommendations to address this issue.

Key Features

  • Survey data collection and analysis
  • Data cleaning and processing
  • Insights into behavioral patterns based on age and demographics

Technologies Used

KoboToolBox, Excel, PowerBI

Insights and Outcomes

The study found a strong negative correlation between excessive smartphone use and productivity. Recommendations included strategies for limiting screen time and promoting healthier usage habits.
